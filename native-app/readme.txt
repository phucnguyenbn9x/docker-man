------ Installation ------

* Environment Requirement
- Install Docker CE:
	https://docs.docker.com/docker-for-windows/install/
- After Install, Goto Setting Docker:
	+ Enable Expose daemon on tcp://localhost:2375
	+ Shared Drives: checkbox C://
	+ Daemon: Enable Basic, add "118.70.171.246:30555" into insecure-registries
- Finnally, Restart Docker

Run as administrator file setup Docker Man.exe
